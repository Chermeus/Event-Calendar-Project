# Застосунок Календар Подій - Опис програми

## Огляд
Календар Подій - це веб-додаток для планування, розроблений для допомоги користувачам організовувати щоденні події, зустрічі та завдання в інтуїтивному календарному форматі. Цей додаток був створений як курсова робота для демонстрації сучасних навичок веб-розробки з використанням HTML, CSS та JavaScript.

## Мета та завдання
**Основна мета:** Надати користувачам простий, але ефективний інструмент для особистого управління часом та організації подій.

**Цільова аудиторія:** Студенти, професіонали та всі, кому потрібно планувати та відстежувати щоденну активність.

**Ключові переваги:**

- Усуває потребу в паперових календарях
- Надає візуальне представлення розкладів
- Дозволяє швидке створення та редагування подій
- Пропонує категоризацію для кращої організації

## Основна функціональність

### Управління подіями
- **Створення подій:** Користувачі можуть додавати нові події, клікнувши на будь-який день у календарі
- **Редагування подій:** Існуючі події можна змінювати через модальні діалоги
- **Видалення події:** Події можна видаляти з підтвердженням
- **Перегляд подій:** Всі події відображаються безпосередньо в календарній сітці

### Навігація по календарю
- **Навігація по місяцях:** Користувачі можуть переглядати різні місяці за допомогою кнопок зі стрілками
- **Виділення поточної дати:** Поточна дата візуально виділена
- **Адаптивна сітка:** Макет 7×5 для відображення всіх днів місяця

### Організація подій
- **Категорії:** Події можуть бути класифіковані як Робота, Особисте або Важливо
- **Кольорове кодування:** Кожна категорія має свій колір (синій, зелений, червоний відповідно)
- **Візуальні індикатори:** Події з'являються як кольорові блоки в клітинках календаря

## Технічна реалізація

### Архітектура
- **Frontend-only рішення:** Не потребує серверних компонентів
- **Об'єктно-орієнтований дизайн:** Побудований з використанням JavaScript ES6 класів
- **Компонентна структура:** Окремі класи для різних функціональностей

### Ключові класи
- **CalendarApp:** Головний контролер додатку
- **CalendarEvent:** Модель даних події та операції з нею

### Зберігання даних
- **localStorage:** Збереження даних на основі браузера
- **Формат JSON:** Події зберігаються як структуровані JSON об'єкти
- **Автоматичне збереження:** Зміни негайно зберігаються в localStorage

### Користувацький інтерфейс
- **Модальні діалоги:** Використовуються для створення та редагування подій
- **Адаптивний дизайн:** Працює на настільних та мобільних пристроях
- **Інтуїтивні елементи управління:** Зрозумілі кнопки та взаємодії

## Функції

### Основні функції
- Створення, читання, оновлення та видалення подій
- Навігація між місяцями
- Категоризація подій за кольорами
- Постійне зберігання даних

### Користувацький досвід
- Чистий, сучасний дизайн інтерфейсу
- Швидке завантаження та швидкий відгук
- Не потребує встановлення
- Працює офлайн після першого завантаження

### Технічні функції
- Кросбраузерна сумісність
- Мобільно-адаптивний макет
- Обробка помилок та валідація
- Чиста, підтримувана структура коду

## Технологічний стек
- **HTML5:** Семантична розмітка та структура
- **CSS3:** Сучасна стилізація з Flexbox та Grid
- **JavaScript (ES6):** Об'єктно-орієнтоване програмування
- **localStorage API:** Збереження даних на стороні клієнта

## Переваги

### Для користувачів
- **Простота:** Легко вивчати та використовувати
- **Доступність:** Працює на будь-якому пристрої з веб-браузером
- **Надійність:** Не потребує залежностей від сервера або інтернету для основного використання
- **Налаштування:** Категоризація подій за кольорами для кращої організації

### Для розробників
- **Чистий код:** Добре структурована, документована база коду
- **Сучасні стандарти:** Використовує сучасні найкращі практики веб-розробки
- **Розширюваність:** Легко додавати нові функції або змінювати існуючі
- **Освітня цінність:** Демонструє практичне застосування веб-технологій

## Висновок

Застосунок Календар Подій успішно демонструє розробку функціонального веб-інструменту для планування. Він поєднує сучасні веб-технології з практичними потребами користувачів для створення ефективного рішення для особистого та професійного управління часом. Додаток служить як корисним інструментом, так і освітнім прикладом принципів та практик веб-розробки.
